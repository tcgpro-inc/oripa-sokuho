# オリパ速報 - 企画書

## 1. サイトコンセプト

**「オリパ速報」** は、ポケモンカード・遊戯王などのトレーディングカードゲーム（TCG）のオリジナルパック（オリパ）に関する最新情報を、昔の2ちゃんねる/なんJ風のレトロなUIで速報配信するニュースアグリゲーションサイト。

### キャッチコピー
> 「オリパの最新情報、誰よりも早く。」

### サイトの特徴
- **速報性**: PRTIMES・X（旧Twitter）などからオリパ関連情報を収集し、速報として配信
- **レトロUI**: 2ch/なんJ風の懐かしいデザインで、TCGユーザーに親しみやすい雰囲気
- **コミュニティ感**: 掲示板風のレイアウトで、ユーザー同士の情報共有を促進

---

## 2. ターゲットユーザー

### メインターゲット
- **年齢層**: 20〜40代男性
- **属性**: ポケカ・遊戯王などのTCGオリパ利用者
- **行動特性**:
  - 新しいオリパ情報を常にチェックしている
  - X（Twitter）でオリパ関連の情報を追っている
  - オリパの当たり報告や評判を気にしている
  - 複数のオリパサイトを比較している

### サブターゲット
- オリパに興味はあるが、まだ利用したことがない初心者層
- オリパ事業者（自社サービスの評判チェック）

---

## 3. 情報ソース

| ソース | 収集内容 | 頻度 |
|--------|----------|------|
| PRTIMES | 新規オリパサービスのプレスリリース、キャンペーン情報 | 日次 |
| X（Twitter） | オリパ関連の話題・当たり報告・炎上情報 | リアルタイム |
| オリパ公式サイト | 新商品・キャンペーン・価格変更 | 日次 |
| YouTube | オリパ開封動画・レビュー | 週次 |
| その他ニュースサイト | TCG市場動向・規制情報 | 随時 |

---

## 4. 主要機能

### MVP（初期リリース）
1. **速報記事一覧** - 最新のオリパ関連ニュースをスレッド形式で表示
2. **カテゴリ分類** - ポケカ / 遊戯王 / ワンピース / その他
3. **記事詳細ページ** - ソース元リンク付きの速報記事
4. **勢いランキング** - 注目度の高い記事をランキング表示
5. **タグ機能** - オリパサービス名・カードゲーム名でのフィルタリング

### Phase 2（将来実装）
6. **オリパ比較表** - 主要オリパサービスの還元率・評判比較
7. **コメント機能** - 記事へのユーザーコメント（匿名）
8. **通知機能** - 気になるキーワードの速報通知
9. **オリパカレンダー** - 発売予定・キャンペーン期間一覧
10. **ユーザー投稿** - 当たり報告・レビュー投稿

## 5. ページ構成

```
/                       ... トップページ（速報一覧 + 勢いランキング）

```

## 6. UIデザイン方針

### コンセプト: 「令和の2ちゃんねる」

| 要素 | 方針 |
|------|------|
| **フォント** | MS Pゴシック / モノスペース系フォント |
| **背景色** | #EFEFEF（グレー系）、#F0E0D6（クリーム系） |
| **テキスト色** | #000000（黒）、#CC1105（赤・重要情報） |
| **リンク色** | #0000FF（青）、#660099（訪問済み） |
| **ボーダー** | 1px solid のシンプルな線 |
| **角丸** | なし（すべて直角） |
| **シャドウ** | なし |
| **レイアウト** | テーブルベース風、高密度テキスト |
| **装飾** | AA（アスキーアート）風のアイコン・区切り |

## 11. 競合分析

| サイト | 特徴 | オリパ速報の差別化ポイント |
|--------|------|--------------------------|
| 各オリパ公式サイト | 自社情報のみ | 横断的な情報集約 |
| X（Twitter） | リアルタイムだが散在 | 整理された速報形式 |
| まとめサイト | 一般TCG情報が中心 | オリパ特化 |
| YouTube | 動画中心で時間がかかる | テキスト速報で即座に把握 |
