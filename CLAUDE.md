# オリパ速報 - CLAUDE.md

## プロジェクト概要
TCGオリパ（ポケカ・遊戯王等）の最新情報を速報配信するニュースアグリゲーションサイト。
2ch/なんJ風のレトロUIが特徴。

## ディレクトリ構成
```
docs/
  企画書.md                  # サイト企画書（コンセプト・機能要件・KPI等）
  design/
    oripa-sokuho-design.pen  # Pencilデザインファイル（全6画面）
    exports/                 # PNG書き出し
```

## デザイン（決定済み）
Pencilデザインファイル: `docs/design/oripa-sokuho-design.pen`

**採用パターン: 「Top Page v2」（jin115参考 + 2ch/なんJ風）**
- ベース: Original（えんじ色ヘッダー、交互クリーム/グレー背景、モノスペースフォント、直角ボーダー）
- jin115.comから取り入れた要素:
  - サムネ付き記事リスト（左画像+右テキスト）
  - ヘッドラインカード（横並び4枚、コメ数バッジオーバーレイ）
  - カテゴリタブ（ヘッダー下の色付きタブナビ）
  - オレンジ色「続きを読む」ボタン
  - 日時+「カテゴリ：」表示（カテゴリはオレンジ色）
  - 記事タイトルは黒太字
- レトロ要素:
  - ★お知らせ★ティッカー（黒背景）
  - ◆マーク装飾、━━罫線
  - 訪問者カウンター（緑文字on黒背景）
  - NEW!/HOT!バッジ
  - 「あなたは○○人目の訪問者です」フッター

※旧パターン（A〜D）もPencilファイル内に残存

## 技術スタック（想定）
- Next.js (App Router)
- Vercel
- Neon (PostgreSQL)
- Tailwind CSS
- Cron Jobs（PRTIMES/X情報収集の自動化）

## ターゲット
20〜40代のTCGオリパ利用者（ポケカ・遊戯王中心）

## コンテンツ分類
記事タグ: 【速報】【朗報】【悲報】【炎上】【まとめ】
カテゴリ: ポケカ / 遊戯王 / ワンピース / その他

## 開発ルール
- UIは「昔の2ちゃんねる/なんJ風」のレトロ感を保つこと
- モノスペースフォントを基本とする（IBM Plex Mono等）
- 角丸は最小限に（採用パターンによる）
- 配色は目に優しい調整済みカラーを使うこと（純粋な #0000FF は避ける）
